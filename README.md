Cybersecurity Awareness Chatbot GUI

The Cybersecurity Awareness Chatbot is a desktop application developed using C# and Windows Presentation Foundation (WPF). The application was developed as part of a second-year Software Development module and is designed to educate users about cybersecurity through interactive conversations, quizzes, reminders, and task management. It helps users learn, remember, and apply essential cybersecurity practices in an engaging and user-friendly environment.

Features

• Greets the user with a voice greeting and ASCII banner.
• Answers cybersecurity-related questions on topics such as phishing, malware, firewalls, password security, ransomware, and data breaches.
• Provides cybersecurity definitions, explanations, and helpful tips.
• Includes a 10-question cybersecurity quiz with feedback.
• Stores the user's favourite cybersecurity topic and recalls it when requested.
• Detects user sentiment and provides encouraging responses where appropriate.
• Allows users to create, view, complete, and delete cybersecurity-related tasks.
• Supports reminders for tasks, such as "remind me in 3 days."
• Maintains an activity log of recent user interactions.
• Uses simulated Natural Language Processing (NLP) to understand different ways of asking similar questions.

Getting Started

1. Clone or download the repository.
2. Open the solution (.sln) file in Visual Studio 2022 or later.
3. Build the project.
4. Ensure the chatbot greeting audio file is located at:

C:\Users\lab_services_student\Music\chatbot.wav

You may replace this file with your own WAV file if preferred.

5. Run the application using MainWindow.xaml.

How to Use the Application

• Enter your name when the application starts.
• The chatbot will greet you and display instructions.
• Ask cybersecurity-related questions.
• Take the cybersecurity quiz.
• Add and manage cybersecurity tasks.
• Set reminders for important tasks.
• View your activity history.
• Ask the chatbot for cybersecurity tips or explanations whenever needed.

Example Commands

• What is phishing?
• Give me a cybersecurity tip.
• Tell me more.
• My favourite topic is malware.
• What is my favourite topic?
• Start quiz.
• Add task change password - update your main account password - remind in 2 days.
• Show tasks.
• Complete task 1.
• Delete task 2.
• What have you done for me?
