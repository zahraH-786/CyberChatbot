Cybersecurity Awareness Chatbot GUI
This is a desktop application designed to teach users about cybersecurity in a fun and interactive way. It uses a smart chatbot that helps users learn, remember, and act on key cyber safety principles.

What It Can Do
Greets the user with a voice and ASCII banner
Answers questions about cybersecurity topics like phishing, malware, firewalls, etc.
Gives definitions and helpful tips based on the topic
Lets you take a quiz with 10 questions and feedback
Stores your favorite topic and reminds you of it when needed
Detects tone (like if you're frustrated) and replies with encouragement
Lets you add, view, delete, and complete cybersecurity-related tasks
Allows reminders for tasks (like “remind me in 3 days”)
Tracks your last few actions in an activity log
Supports different ways of asking the same thing (NLP simulated)

How to Set It Up
Clone or download the repository
Open the .sln solution file in Visual Studio
Build the project
Make sure the chatbot.wav file is in your Music folder at:
C:\Users\lab_services_student\Music\chatbot.wav
This is used for the greeting sound. You can replace it with your own WAV file if needed
Run the application using MainWindow.xaml

How to Use It
When the app opens, enter your name
The chatbot will greet you and give you instructions

Try asking:
What is phishing
Give me a tip
Tell me more
My favourite topic is malware
What is my favourite topic
Start quiz
Add task change password - update your main account password - remind in 2 days
Show tasks
Complete task 1
Delete task 2
What have you done for me
